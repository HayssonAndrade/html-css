Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase tenho um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica).

Nesta frase temos um termo em itálico usando a tag EM (Semântica)

Texto marcado

Podemos criar tanbém um texto marcado usando a tag MARK.

Texto excluído / Inserido depois

Podemos marcar um texto como exclído / Não considerar usando a tag DEL.

Podemos marcar um texto como inserido depois usando a tag INS.

Texto sobrescrito / Texto subscrito

Podemos destacar algo sobrescrito usando a tag SUP. EXP: x20+3

Tanbém podemos fazer o contrário para subscrever algo, usando a tag SUB. EXP: H2O
